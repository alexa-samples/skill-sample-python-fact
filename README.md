## What Your Skill Will Do
This simple "quiz" Skill will teach you how the different pieces of the Alexa Skill development process fit together. The Skill allows users to request a quiz about the 50 States of the USA to which they will receive 10 random questions or ask questions, such as, "tell me about New York".

We use will use the new Skill builder UI to build the Skill although the quiz itself does not employ the new "Dialog-Delegate interface" model.
